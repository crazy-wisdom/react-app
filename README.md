# react-app

The purpose aims to a boilerplate template to run react application quickly.

#### Features
- typescript
- redux
- react-route
- bootstrap
- fontawesome
- sass
- snowpack
- jest


### Create env files
```
cp .env.example .env.local
cp .env.example .env.test
```

#### Install

```sh
npm install
```

#### Start

```sh
npm start
```

Then go to http://localhost:8080


```sh
npm run build
```
